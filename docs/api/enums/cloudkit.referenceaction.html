DELETE_SELF

DELETE_SELF:

No action when a referenced record is deleted. The deletion of the additional records may trigger a cascade deletion of more records. The deletions are asynchronous in the default zone and immediate in a custom zone.

NONE

NONE:

Deletes a source record when the target record is deleted. Deleting a parent record does not delete the children that refer to that parent.
