Optional continuationMarker

continuationMarker: undefined | null | string
  • Defined in tsl-apple-cloudkit.d.ts:2409

Marks the location of the last batch of results. Use this key when the results of a previous fetch exceed the maximum. The default value is null.

Optional desiredKeys

desiredKeys: undefined | null | string[]
  • Defined in tsl-apple-cloudkit.d.ts:2417

An array of strings containing record field names that limits the amount of data returned in this operation. Only the fields specified in the array are returned. The default is null, which fetches all record fields.

Optional numbersAsStrings

numbersAsStrings: undefined | false | true
  • Defined in tsl-apple-cloudkit.d.ts:2423

A Boolean value indicating whether number fields should be represented by strings. The default value is false.

Optional resultsLimit

resultsLimit: undefined | number
  • Defined in tsl-apple-cloudkit.d.ts:2429

The maximum number of records to fetch. The default is the maximum number of records in a response that is allowed.

Optional zoneID

zoneID: undefined | string | ZoneID
  • Defined in tsl-apple-cloudkit.d.ts:2436

A ZoneID or zone name that identifies the record zone in the database where you want to perform the operation. The default is the database default zone.

Optional zoneWide

zoneWide: undefined | false | true
  • Defined in tsl-apple-cloudkit.d.ts:2443

Boolean value determining wether all zones should be searched. This key is ignored if zoneID is non-null. To search all zones, set to true. To search the default zone only, set to false.
